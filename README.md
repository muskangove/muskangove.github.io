**Personal Portfolio Website – Muskaan Gove**

1. Introduction:

A personal portfolio website is a digital platform used to showcase an individual’s professional profile, skills, projects, and contact details. This project represents my personal portfolio developed using HTML, CSS, JavaScript, and Bootstrap to present my educational background, technical skills, and project work in a structured and visually appealing manner.

2. Objectives

- To create a professional online presence
- To showcase skills in Power BI, Python, Web Development
- To display academic details and project work
- To provide easy access to contact information and resume
- To practice front-end web development concepts

3. Technologies Used

Technology	                                                                       Purpose

- HTML5 Structure of the website
- CSS3 Styling, layout, animations
- JavaScript Typing animation, interactivity
- Bootstrap Responsive design and layout
- jQuery DOM manipulation and animations
- Owl Carousel Slider functionality
- AOS (Animate on Scroll) Scroll-based animations
- Google Fonts Custom typography
- GitHub Pages Hosting the portfolio

4. Website Structure

The website consists of the following main sections:

4.1 Home Section

- Displays name and professional title
- Animated typing effect showing roles:
- Fresh Graduate
- Engineer
- Data Analyst Intern
- Call-to-action button linking to GitHub projects

4.2 About Section

1. Profile image and personal details

2. Overview of education, experience, and location

3. Skill set displayed using progress bars:

- Power BI
- Data Visualization
- Python
- HTML
- CSS
- JavaScript

4. Short professional summary

4.3 Resume Section

- Education details (Bachelor of Engineering – VTU)
- Downloadable CV via Google Drive link

4.4 Projects Section

Displays real-world projects with live and GitHub links:

1. Weather Information Website

- Built using HTML, CSS, and JavaScript
- Displays real-time weather details

2. Power BI Sales Dashboard

- Sales analysis and profit tracking across India
- Interactive data visualization using Power BI

4.5 Contact Section

- Address (Bangalore, India)
- Phone number
- Email ID
- Resume download link

5. Key Features

- Fully responsive design
- Smooth scrolling navigation
- Animated typing text
- Hover zoom effects on project cards
- Skill progress indicators
- Clean UI and modern layout
- External links to GitHub, LinkedIn, Power BI

6. Code Highlights

1. Typing Animation: Implemented using JavaScript with dynamic text switching

2. Progress Bars: Visual representation of skills using Bootstrap

3. Carousel Slider: Used Owl Carousel for homepage slider

4. Animations: AOS library enhances user engagement

5. Modular Design: Clean separation of HTML, CSS, and JS files

7. Challenges Faced

- Creating smooth typing animation without glitches
- Ensuring responsive design on different screen sizes
- Managing multiple JS libraries together
- Optimizing layout alignment for different sections

8. Future Enhancements

- Add backend contact form (Node.js / PHP)
- Improve accessibility (ARIA labels)
- Add certifications section
- Dark/Light theme toggle
- SEO optimization

9. Conclusion

This personal portfolio website successfully demonstrates my skills in front-end web development and data analytics visualization. It serves as a professional platform to present my projects, technical expertise, and academic background. The project helped me strengthen my understanding of responsive design, animations, and real-world UI development.
